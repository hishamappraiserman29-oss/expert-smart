📘 دليل Expert_Smart v37

النظام يجمع بين قوة Gemini AI وتقنية الرصد الآلي اللحظي لنبض السوق لضمان دقة تحاكي الواقع لحظة بلحظة

📂 رفع الملفات — مسار مزدوج
  • نموذج المحاكاة — ارفع تقريراً مرجعياً (PDF/Excel) ليتعلّم النظام أسلوبه
  • بيانات العقار — ارفع البيانات الخام (عقد، رخصة، مخطط، إحصاءات)
⚙ بوابة المسار الذكي (Logic Gate)
  • اختر نوع الأصل (سكني / تجاري / صناعي / زراعي / فندقي / تجزئة / صحي / تعليمي)
  • اختر الغرض (قيمة سوقية / تمويل بنكي / استحواذ / استثماري / تحكيم / تأميني / قضائي)
  • يتكيف التقرير الإكسيل تلقائياً بـ 20 شيتاً يُحذف غير المناسب منها وفق المسار
🤖 المستشار العقاري الذكي (RAG)
  • يعمل بتقنية RAG على قاعدة بيانات السوق المصري المحلية + البحث على الإنترنت
  • اسأله عن أسعار أي منطقة، تأثير الأحداث الجيوسياسية، أفضل فرص الاستثمار
  • ✅ يعمل بدون مفتاح API — يستخدم نموذج Qdrant + SentenceTransformer المحلي
📚 محرك المحاكاة وضبط الأسلوب
  • ارفع تقريراً مرجعياً → يُحلَّل أسلوبه ومنهجيته → يُحفظ في المكتبة كـ Profile
  • فعِّل Profile → تُحقَن بصمة الأسلوب في كل استفسار تالٍ لـ Gemini
  • المكتبة تُصفِّي المكررات وتحتفظ بالنماذج المميزة فقط
🛰 رادار السوق الآلي (v37)
  • يرصد Aqarmap + Bayut + OLX تلقائياً كل 30 دقيقة
  • يُحوِّل النصوص العقارية غير المنظَّمة (واتساب/مواقع) إلى جداول بيانات منظَّمة
  • يُغذِّي خريطة مناطق الأسعار (Heatmap) وتقارير IAAO بيانات حية
📊 معايير IAAO للمسار الضريبي
  • ASR (0.95–1.05): نسبة التقييم إلى البيع — إشارة مرور خضراء/صفراء/حمراء
  • COD (<15%): معامل التشتت — العدالة الأفقية بين الممتلكات
  • PRD (0.98–1.03): مؤشر التفاوت السعري — العدالة الرأسية
  • يُدمَج تلقائياً في تقرير الإكسيل عند اختيار مسار "مسؤول ضريبي"
🔑 مفتاح Gemini API
  • احصل على مفتاحك من ai.google.dev (مجاناً)
  • اضغط زر 🔑 في الرأس أو فعِّله من هنا
  • يُحفظ في المتصفح محلياً فقط — لا يُرسَل لأي خادم خارجي
